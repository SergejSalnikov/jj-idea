# Roadmap

Planned features, roughly in priority order.

1. **Remote Bookmark Management** — Track/untrack remote bookmarks, handle conflicted bookmarks, branches panel, bookmark toolbar widget
2. **Squash & Split from File Selection** — Squash/split individual files from context menus, destination picker, hunk-level granularity
3. **Conflict Resolution** — Dedicated UI for resolving merge conflicts
4. **Multi-Select in Log** — Show combined details and changes when multiple commits are selected
5. **Open in Remote** — Jump from file history to GitHub/GitLab in one click
6. **Operation Log & Undo** — Browse `jj op log`, undo/redo operations with toolbar buttons
7. **Remote Management** — Add/remove/rename Git remotes, provider-aware clone with authentication
8. **JJ-Unique Operations** — Evolog (change evolution), duplicate, absorb, interdiff, merge signposting
9. **Forge Integration** — Create and view GitHub/GitLab pull requests and merge requests from the IDE

Have a feature request? [Open an issue](https://github.com/kkkev/jj-idea/issues).
